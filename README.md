# RAG-Based Question Paper Generator

A Flask-based AI application that generates question papers from uploaded PDF notes using Retrieval-Augmented Generation (RAG).

## Features
- Upload PDF notes
- Extract text from PDF
- Split into chunks
- Create FAISS vector database
- Retrieve relevant content
- Generate question paper using AI
- MCQs, Short and Long Questions
- Answer Key

## Tech Stack
- Python
- Flask
- FAISS
- Sentence Transformers
- Google Gemini API
- PyPDF

## Future Improvements
- PDF Question Paper Download
- Difficulty Levels
- Bloom's Taxonomy
- Multi-language Support
